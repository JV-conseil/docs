Recommended VS Code extensions in `~/GitHub/<owner>/<repo>/.vscode/extensions.json` 👇

```json

  ]
}
```

Review your VS Code settings in 👇

1. `~/GitHub/<owner>/<repo>/.vscode/settings.json`
